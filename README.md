# Date Dime

Date Dime is a quick and easy way to find a restaurant that fits your budget and what's in theaters!

## Details

- Here's a quick link to try it yourself:
  https://lyndsclev.github.io/date-dime/

## Technology Used

- HTML
- Bulma
- CSS
- JavaScript
- JQuery
- API's: MovieGlu, GoogleMaps
